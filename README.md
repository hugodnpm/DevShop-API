# DevShop API

## Description

Projeto de API do DevShop (Backend)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Database Migrations

```bash
# generate a new migration synchronizing Entities
$ npm run typeorm migration:generate -- -n <migration description>

# sync database running migrations
$ npm run typeorm migration:run


```
